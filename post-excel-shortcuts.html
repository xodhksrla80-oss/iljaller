💡 단축키, 왜 30개만?

엑셀 단축키는 공식적으로 500개 이상이에요. 그런데 실무에서 진짜 쓰는 건 30개도 안 돼요. 나머지는 알아도 까먹고, 쓸 일도 별로 없어요.

⚡
하루에 2~3개씩만 익혀도 충분해요

오늘 Ctrl+E 하나만 써도 이미 이득이에요. 한꺼번에 외우려 하지 말고, 하나씩 손에 익히는 게 훨씬 오래 가요.

🔑 기본 필수 10개 — 지금 당장

엑셀을 처음 시작했어도 이것만 알면 기본은 한다는 소리 들어요.

단축키기능효과
Ctrl+S 저장필수
Ctrl+Z 실행 취소필수
Ctrl+C / Ctrl+V 복사 / 붙여넣기필수
Ctrl+; 오늘 날짜 자동입력필수
F2 셀 편집모드 진입추천
F4 마지막 작업 반복필수
Ctrl+F 찾기필수
Ctrl+H 찾아바꾸기필수
Ctrl+1 셀 서식 창 열기추천
Alt+F4 엑셀 종료추천
🔥
F4가 생각보다 엄청나요

셀 색상을 바꾼 뒤 다른 셀에서 F4 누르면 같은 작업이 반복돼요. 테두리, 서식, 행 삽입 모두 적용돼요. 마우스 클릭 횟수가 확 줄어요.

📊 데이터 작업 10개 — 속도 3배

데이터가 많은 날, 이것만 쓰면 마우스를 거의 안 써도 돼요.

단축키기능효과
Ctrl+↑↓←→ 연속 데이터 끝으로 이동필수
Ctrl+Shift+↓ 끝까지 범위 선택필수
Ctrl+E 빠른 채우기 (패턴 자동인식)필수
Ctrl+T 표 만들기추천
Ctrl+Shift+L 필터 켜기/끄기추천
Alt+= 자동 합계(SUM)필수
Ctrl+D 아래 셀 채우기추천
Ctrl+Home A1셀로 바로 이동추천
Ctrl+End 데이터 마지막 셀로 이동추천
Alt+Enter 셀 안에서 줄바꿈추천
🚀
Ctrl+E 이거 진짜 마법이에요

예: A열에 "홍길동(010-1234-5678)" 형식으로 되어 있을 때, B1에 "홍길동"만 입력하고 B2에서 Ctrl+E 누르면 나머지를 자동으로 채워줘요. 이름·번호·이메일 분리에 최고예요.

📄 서식·보고서 10개 — 마감 전 필수

보고서 마감 전날, 이것만 알아도 30분은 단축돼요.

🎯 손에 익히는 법

1
오늘 딱 3개만 골라서 써보세요

추천 첫 세트: Ctrl+E, F4, Ctrl+; — 이 세 개만으로도 하루 10분은 아낄 수 있어요.

2
마우스 손이 가려는 순간 참고 키보드로

처음엔 어색해도 2~3일만 버티면 손이 먼저 움직여요. 불편함을 참는 게 핵심이에요.

3
모니터 옆에 프린트해서 붙여두세요

이 글을 인쇄해서 모니터 옆에 붙여두면 일주일 안에 자연스럽게 외워져요.

🎯 핵심 요약
  • F4 — 마지막 작업 반복. 서식·삽입 모두 적용돼요
  • Ctrl+E — 빠른 채우기. 이름·번호 분리에 마법 같은 기능
  • Ctrl+방향키 — 대용량 데이터에서 끝으로 순식간에 이동
  • Alt+= — SUM 함수 타이핑 없이 자동 합계
  • 하루 3개씩 2주 = 엑셀 단축키 마스터
