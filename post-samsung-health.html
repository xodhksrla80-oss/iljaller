📱 폰만 있어도 되는 기능 6가지

🚶 걸음수 자동 측정
앱 실행 안 해도 폰을 들고 다니면 자동으로 걸음수·이동 거리·소모 칼로리 기록. 목표 걸음수 설정하면 달성 알림도 와요.
📌 홈 → 걸음수 카드 → 일일 목표 설정
😴 수면 분석 (폰으로도 가능)
폰을 침대 위 또는 매트리스 근처에 두면 수면 시간·수면 단계를 분석해줘요. 갤럭시 워치만큼 정확하진 않지만 패턴 파악엔 충분해요.
📌 수면 → 수면 측정 → 폰 감지 모드 ON
💓 심박수 측정
카메라에 손가락을 올리면 심박수 측정 가능. 워치 없이도 스트레스 많은 날 심박수 확인하는 용도로 쓸 수 있어요.
📌 홈 → 심박수 → 측정 시작
🍽️ 식단 기록 & 칼로리 계산
먹은 음식 검색하거나 바코드 스캔으로 칼로리 자동 입력. 국내 음식 DB가 잘 구축되어 있어서 한식도 잘 찾혀요.
📌 식단 → + 버튼 → 음식 검색 또는 바코드 스캔
🏃 운동 기록 자동화
걷기·달리기·자전거 등 시작하면 GPS 경로·속도·칼로리 자동 기록. 실내 운동도 40여 종 지원해요.
📌 운동 → 운동 시작 → 종류 선택
🏆 Together 챌린지
친구·동료와 걸음수 경쟁하는 챌린지 기능. 팀 단위로 걸음수 합산해서 경쟁도 가능. 회사 동료들과 건강 챌린지하기 딱이에요.
📌 Together → 챌린지 → 친구 초대

⌚ 갤럭시 워치 연동 시 추가 기능

⌚
워치 있으면 이게 더 추가돼요

• 수면 단계 정밀 분석 (렘·깊은 수면·얕은 수면 구분)
• 혈중 산소 포화도(SpO2) 측정
• 피부 온도 모니터링
• 심전도(ECG) 측정 (Galaxy Watch 4 이상)
• 스트레스 지수 실시간 자동 측정

🗓️ 직장인 건강 루틴 세팅법

✅
이 3가지만 세팅해두면 됩니다

① 일일 걸음수 목표 8,000보 설정 → 달성 알림 ON
② 수면 목표 시간 설정 (권장 7~8시간) → 취침·기상 리마인더
③ 물 마시기 목표 설정 → 하루 8잔 알림

💡
구글 피트니스·애플 헬스와 연동도 됩니다

삼성 헬스 데이터를 구글 피트니스로 내보내기 가능해요. 아이폰 유저는 갤럭시 폰 없이는 사용 불가하지만, 갤럭시 폰 사용자라면 기본 탑재되어 있어요.

🎯 핵심 요약
  • 삼성 헬스는 갤럭시 폰만 있으면 걸음수·수면·심박수·식단 모두 무료 측정
  • Together 챌린지로 회사 동료들과 걸음수 경쟁 가능
  • 갤럭시 워치 연동 시 혈중산소·피부온도·ECG까지 추가
  • 일일 걸음수 목표 + 수면 목표 + 물 마시기 3가지만 세팅해도 충분
  • 설치만 하고 방치하는 분 많음 — 오늘 알림 설정만 해도 달라져요
