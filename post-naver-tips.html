🟢 그린닷 — 화면 어디서든 네이버 검색

그린닷은 스마트폰 화면 어디서든 떠다니는 네이버 검색 버튼이에요. 다른 앱 쓰다가 앱 전환 없이 바로 검색할 수 있어요.

켜기
네이버 앱 → 설정 → 그린닷 켜기

켜면 화면 가장자리에 초록 점이 생겨요. 언제든 탭하면 검색창이 뜹니다.

활용
텍스트 선택 → 그린닷으로 바로 검색

웹페이지나 카톡에서 모르는 단어 선택 → 그린닷 탭 → 앱 전환 없이 검색 결과 확인.

💡
번역도 바로 돼요

영어 텍스트 선택 → 그린닷 → 파파고 번역 바로 가능. 영문 계약서나 이메일 볼 때 앱 전환 없이 번역할 수 있어요.

🎙 클로바 노트 — AI가 회의록 자동 작성

클로바 노트는 음성을 텍스트로 변환해주는 AI 서비스예요. 회의 녹음하면 자동으로 내용 정리까지 해줍니다.

1
네이버 앱 → 클로바 노트 or 별도 앱 설치

CLOVA Note 앱 또는 네이버 앱 내 클로바 노트 접속. 네이버 계정으로 로그인.

2
녹음 시작 → AI가 실시간 텍스트 변환

회의 중 녹음 버튼만 누르면 돼요. AI가 화자를 구분하고 발언 내용을 텍스트로 정리해줍니다.

3
AI 요약 기능으로 핵심만 추출

긴 회의도 AI가 핵심 내용만 요약. 회의록 작성 시간이 90% 줄어요.

✅
무료로 월 300분 사용 가능

클로바 노트 무료 플랜으로 월 300분 녹음·변환 가능. 주 1~2회 회의라면 무료로 충분해요.

⌨️ 스마트보드 — 키보드 위의 네이버

스마트보드는 네이버 전용 키보드예요. 타이핑하다가 키보드 위에서 바로 검색·번역·지도를 사용할 수 있어요.

설치
네이버 스마트보드 앱 설치 → 기본 키보드 설정

앱 설치 후 스마트폰 키보드 설정에서 네이버 스마트보드로 변경. 10초면 완료.

활용
키보드 상단 바에서 검색·번역·이미지 검색

카톡 대화 중에도 키보드 상단에서 바로 검색 결과 공유 가능. "맛집 찾아서 링크 공유"가 앱 전환 없이 가능.

📷 렌즈 — 카메라로 뭐든 검색

네이버 앱 검색창 옆 카메라 아이콘이 렌즈예요. 사진 찍으면 뭔지 바로 알려줘요.

🌸
식물·꽃 인식

꽃이나 식물 사진 찍으면 이름과 특징을 알려줘요. 봄철 꽃 사진 찍을 때 딱.

🍽
음식 칼로리 확인

음식 사진 찍으면 예상 칼로리와 영양 정보 제공. 다이어트 중이라면 필수.

📄
문서·명함 텍스트 인식(OCR)

명함 찍으면 연락처 자동 저장. 종이 문서 찍으면 텍스트로 추출. 손으로 옮겨 적을 필요 없어요.

⚡ 추가 꿀기능 5가지

1
MY구독 — 관심 주제 뉴스만 모아보기

네이버 앱 홈 → MY구독 설정. 관심 언론사·주제만 골라서 피드 구성 가능. 불필요한 뉴스 없이 깔끔하게.

2
네이버 지도 — 저장한 장소 공유

맛집 목록을 "내 장소"에 저장해두면 친구에게 컬렉션 전체를 링크로 공유 가능. 약속 잡을 때 유용.

3
파파고 즉석 대화 통역

파파고 앱 → 대화 모드. 외국인과 번갈아 말하면 실시간 통역. 여행·업무에서 실제로 써보면 놀라워요.

4
네이버 NOW — 실시간 라이브 방송

네이버 앱 → NOW. 연예인·크리에이터 실시간 방송을 네이버 앱 안에서 바로 시청 가능.

5
검색 히스토리 자동 삭제 설정

설정 → 검색 설정 → 최근 검색어 자동 삭제 주기 설정. 공용 기기 쓸 때 프라이버시 보호.
