💡 구글 드라이브 기본 정리

구글 드라이브는 구글 계정만 있으면 무료로 15GB를 쓸 수 있어요. 단순 저장소가 아니라 Google Docs, Sheets, Slides까지 포함된 오피스 패키지예요.

📊
2026년 기준 구글 드라이브 현황

전 세계 10억 명 이상 사용, 매일 30억 개 이상 파일 업로드. Word·Excel·PPT 파일도 바로 편집 가능하고, Gemini AI가 문서 요약까지 해줘요.

Gmail·드라이브·포토가 15GB를 공유해서 써요. Gmail 정리만 해도 용량이 확 늘어납니다.

🔑 몰랐으면 손해인 꿀기능 6가지

1
📄 PDF·이미지에서 텍스트 추출 (무료 OCR)

스캔한 PDF나 이미지 속 텍스트를 복사하고 싶을 때 쓰는 기능이에요. 별도 프로그램 없이 드라이브에서 바로 됩니다.

📌 방법: 드라이브에 PDF 업로드 → 우클릭 → "Google Docs로 열기" → 텍스트 자동 추출!
2
🤖 Gemini AI로 문서 요약

2026년부터 구글 드라이브에 Gemini AI가 통합됐어요. 긴 문서를 열고 Gemini에게 "이 문서 3줄로 요약해줘"라고 하면 바로 됩니다.

📌 방법: Google Docs 열기 → 우측 Gemini 아이콘 클릭 → 질문 입력
3
🔗 공유 링크 만료 날짜 설정

외부에 파일 공유할 때 영구 공개가 걱정되시죠? 특정 날짜 이후 자동으로 접근 불가하게 설정할 수 있어요.

📌 방법: 파일 우클릭 → 공유 → 링크 설정 → 액세스 만료 날짜 지정
4
📴 오프라인에서도 작업 가능

인터넷이 없어도 구글 Docs·Sheets를 편집하고, 인터넷 연결되면 자동 동기화돼요. 비행기 탑승 전에 설정해두면 기내에서도 작업 가능합니다.

📌 방법: 드라이브 설정 → "오프라인에서 파일 보기 및 편집" 켜기
5
📂 바로가기로 중복 없이 여러 폴더 관리

같은 파일을 여러 폴더에 넣고 싶을 때 복사 대신 바로가기를 만들면 용량을 아낄 수 있어요. 파일은 하나, 접근 경로는 여러 개예요.

📌 방법: 파일 우클릭 → "바로가기 추가" → 원하는 폴더 선택
6
🎨 폴더 색상으로 빠른 구분

폴더가 많으면 찾기가 힘들죠. 중요한 폴더에 색상을 지정하면 시각적으로 바로 찾을 수 있어요. 프로젝트별로 색상 구분하면 훨씬 편해요.

📌 방법: 폴더 우클릭 → "색상 변경" → 원하는 색 선택

💾 15GB 용량 아끼는 법

15GB가 금방 찬다고 느끼면 대부분 Gmail이 주범이에요. Gmail 정리만 해도 5~8GB는 금방 확보돼요.

✅
용량 확보 3단계

① Gmail 검색창에 "has:attachment larger:5mb" 입력 → 대용량 메일 삭제
② 구글 포토에서 고화질→절약 화질로 변환 요청
③ 드라이브에서 오래된 임시파일 삭제

💡
용량 부족하면 Google One 100GB

월 2,900원으로 100GB 확장 가능해요. Gmail+드라이브+포토 전부 공유로 쓰니까 가성비 최고예요.
