⚙️ 아이패드 업무 핵심 세팅 3가지

🪟 Split View — 두 앱 동시에 열기
노션 + 사파리, PDF + 메모 앱을 화면 반반씩 띄워서 동시에 작업 가능. 아이패드 업무 활용의 핵심이에요.
📌 앱 열기 → 상단 … 버튼 → Split View → 두 번째 앱 선택
📌 Stage Manager — 멀티태스킹 강화
iPad OS 16 이상에서 지원. 여러 앱을 자유롭게 배치하고 그룹으로 관리. 맥북처럼 창 기반 작업이 가능해요. (M1 이상 아이패드 지원)
📌 설정 → 홈 화면 및 멀티태스킹 → Stage Manager 켜기
⌨️ 외장 키보드 연결
매직 키보드 또는 블루투스 키보드 연결하면 노트북처럼 쓸 수 있어요. Apple 매직 키보드는 트랙패드까지 포함되어 마우스 없이 완전한 작업 환경이 됩니다.
📌 설정 → 블루투스 → 키보드 연결

📱 직장인 필수 앱 5가지

📋
노션 (Notion)
아이패드에서 Apple Pencil로 손글씨 → 텍스트 변환, Split View로 자료 보면서 메모. 회의 중 노트 앱으로 최고.
무료PDF 편집
📄
PDF Expert
계약서·보고서 PDF에 직접 서명·주석·하이라이트 가능. 아이패드 PDF 편집 앱 중 가장 완성도 높아요.
유료 $79.99/년무료 체험 가능
✏️
Goodnotes 6
Apple Pencil로 손글씨 노트. AI가 손글씨를 텍스트로 변환, 검색도 가능. 회의록·아이디어 스케치에 최고.
무료 (기본)Apple Pencil 추천
📊
Microsoft 365 (Word·Excel·PPT)
아이패드에서 Word·Excel·PowerPoint 완전 편집 가능. 회사 문서 작업에 필수. Microsoft 365 구독 있으면 무료.
무료 (기본 편집)M365 구독 시 전체 기능
🤖
ChatGPT / Claude
아이패드 앱으로 출시. 카메라로 문서·칠판 찍으면 내용 분석·요약해줘요. 이동 중 AI 작업에 최적.
무료유료 플랜 추천

🖥️ 맥북·PC 보조 모니터로 쓰기

🖥️
Sidecar — 맥북의 두 번째 모니터

맥북과 아이패드를 함께 쓴다면 Sidecar 기능으로 아이패드를 무선 보조 모니터로 활용 가능. 별도 케이블·앱 불필요. 맥 메뉴바 → 디스플레이 → 아이패드로 미러링/확장 선택.

🪟
Windows PC라면 — Duet Display

윈도우 PC 사용자는 Duet Display 앱으로 아이패드를 보조 모니터로 연결 가능. 유료 앱이지만 외부 모니터 대비 가성비가 좋아요.

어떤 아이패드를 사야 할까?
