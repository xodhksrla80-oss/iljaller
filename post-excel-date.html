🔢 날짜가 숫자로 보이는 이유

엑셀에서 날짜를 입력했는데 45300, 46102 같은 숫자로 표시된다면 당황스럽죠. 엑셀은 날짜를 내부적으로 숫자로 저장합니다. 1900년 1월 1일을 숫자 1로 보고, 하루가 지날 때마다 1씩 올라가는 방식이에요.

엑셀 날짜 원리
📅 2026년 3월 22일 = 숫자 46101
날짜 자체는 정상적으로 저장된 것입니다.
보여주는 형식(셀 서식)만 바꿔주면 바로 해결돼요.

즉, 저장된 데이터 자체는 정상입니다. 셀 서식이 "날짜"가 아닌 "일반" 또는 "숫자"로 되어 있는 게 원인이에요. 서식만 바꿔주면 바로 날짜로 표시됩니다.

⚠️
이런 경우도 있어요

다른 시스템에서 내보낸(export) 엑셀 파일에서 날짜가 숫자로 표시되는 경우도 많아요. 원인은 같아요. 셀 서식을 날짜로 바꿔주면 됩니다.

🎨 셀 서식으로 날짜 형식 바꾸기

숫자로 표시된 날짜 셀을 선택하고 서식만 바꿔주면 됩니다. 두 가지 방법이 있어요.

방법 1 — 빠른 방법 (단축키)

1
날짜로 바꿀 셀 선택

여러 셀이면 드래그로 범위 선택. 열 전체라면 열 머리글(A, B, C)을 클릭.

2
Ctrl + 1 → 셀 서식 창 열기

셀 서식 대화상자가 열립니다. 맥은 Cmd + 1.

3
범주 → 날짜 선택 → 원하는 형식 클릭 → 확인

2026-03-22, 2026년 3월 22일 등 원하는 형식을 고를 수 있어요. 미리보기에서 확인 후 확인 클릭.

방법 2 — 홈 탭에서 바꾸기

①
홈 탭 → 상단 리본의 "표시 형식" 드롭다운

기본적으로 "일반"이라고 표시되어 있는 드롭다운 메뉴입니다.

②
"간단한 날짜" 또는 "자세한 날짜" 선택

간단한 날짜는 2026-03-22 형식, 자세한 날짜는 2026년 3월 22일 형식입니다.

✅
사용자 지정 날짜 형식 만들기

Ctrl+1 → 사용자 지정 범주 선택 → 형식 입력란에 yyyy/mm/dd 또는 yy.mm.dd 등을 직접 입력하면 원하는 형식으로 자유롭게 표시할 수 있어요.

⌨️ 오늘 날짜 자동입력 단축키

매번 오늘 날짜를 직접 입력하는 분들 많죠. 단축키 하나면 즉시 입력할 수 있어요. 두 가지 방식이 있으니 용도에 맞게 골라 쓰세요.

Ctrl + ;
오늘 날짜 고정 입력
(변하지 않는 정적 값)
Ctrl + Shift + ;
현재 시각 고정 입력
(변하지 않는 정적 값)
💡
TODAY() 함수와의 차이

단축키 Ctrl+;로 입력한 날짜는 고정값으로, 파일을 내일 열어도 오늘 날짜 그대로입니다. 반면 =TODAY() 함수는 파일을 열 때마다 자동으로 오늘 날짜로 바뀌어요. 문서 작성일 기록에는 단축키를, 남은 D-Day 계산에는 TODAY() 함수를 쓰세요.

📆 날짜 자동 채우기 (드래그 채우기)

날짜를 연속으로 입력해야 할 때 하나하나 타이핑하면 시간이 너무 걸려요. 엑셀의 자동 채우기 기능을 쓰면 클릭 몇 번으로 해결됩니다.

1
시작 날짜 입력

예: A1에 2026-03-01 입력. 날짜 형식으로 인식되어야 채우기가 작동해요.

2
셀 선택 → 오른쪽 하단 초록 점(채우기 핸들) 드래그

마우스를 올리면 + 모양으로 바뀌는 점. 아래로 드래그하면 날짜가 하루씩 증가해요.

3
채우기 옵션 선택 (오른쪽 하단 아이콘)

드래그 후 나타나는 아이콘을 클릭하면 "일 단위", "평일 단위", "월 단위", "연 단위" 중 선택 가능합니다. 주말 제외한 평일만 채우려면 "평일 단위"를 선택하세요.

🎯
날짜 채우기 안 될 때 확인사항

채우기를 해도 같은 날짜가 반복된다면, 해당 셀이 날짜가 아닌 텍스트로 저장된 것입니다. 셀 왼쪽 정렬로 표시되어 있으면 텍스트예요. Ctrl+1로 서식을 날짜로 바꾼 후 다시 시도해보세요.

🎯 핵심 요약
  • 날짜가 숫자로 보이면 → Ctrl+1 → 날짜 서식 선택. 데이터는 정상, 서식만 바꾸면 됨
  • 오늘 날짜 고정 입력 → Ctrl+; / 현재 시각 → Ctrl+Shift+;
  • 매일 자동 갱신되는 오늘 날짜 → =TODAY() 함수 사용
  • 날짜 자동 채우기 → 시작 날짜 입력 후 채우기 핸들 드래그 → 평일만도 가능
  • 채우기가 안 되면 → 셀이 텍스트로 저장된 것, 서식을 날짜로 변경 후 재시도
