🔤 글꼴 일괄 변경 — 100장 3초

팀원 여러 명이 취합한 PPT를 받으면 폰트가 제각각이에요. 슬라이드 100장을 하나하나 바꾸면 1시간이 걸리지만, 이 기능 하나로 3초면 돼요.

1
글꼴 일괄 변경 (Windows)

홈 탭의 '바꾸기' 메뉴에서 글꼴만 바꾸는 전용 기능이 있어요. 찾기/바꾸기(Ctrl+H)와 다른 기능이에요!

📌 홈 탭 → 편집 그룹 → 바꾸기(▼) 클릭 → [글꼴 바꾸기(C)] → 현재 글꼴 선택 → 새 글꼴 선택 → 바꾸기
2
글꼴 일괄 변경 (Mac)

Mac 버전도 동일한 기능이 있어요. 메뉴 경로만 조금 달라요.

📌 편집 → 찾기 → 글꼴 바꾸기 → 현재 글꼴 / 새 글꼴 선택 → 모두 바꾸기
🔥
Ctrl+H(찾기/바꾸기)랑 다릅니다!

일반 찾기/바꾸기는 텍스트 내용을 바꾸는 것이고, 글꼴 바꾸기는 서식(폰트)을 바꾸는 것이에요. 메뉴에서 '글꼴 바꾸기'를 꼭 찾아서 써야 해요.

🎨 슬라이드 마스터 활용법

슬라이드 마스터를 한 번만 세팅해두면 모든 슬라이드에 회사 폰트·색상·로고가 자동으로 적용돼요. 팀 전체가 일관된 디자인으로 작업할 수 있는 핵심 기능이에요.

3
슬라이드 마스터 설정

회사 표준 폰트, 브랜드 색상, 로고 위치를 한 번에 세팅하고 .potx 파일로 저장해두면 팀 전체가 공유할 수 있어요.

📌 보기 탭 → 마스터 보기 → 슬라이드 마스터 → 서식 편집 → 마스터 보기 닫기
💾 저장 시 파일 형식을 "PowerPoint 서식 파일(.potx)"로 선택
✅
팀 템플릿 공유하는 법

.potx 파일을 구글 드라이브나 공유 폴더에 저장해두면 팀원 모두가 같은 마스터로 작업할 수 있어요. 매번 폰트·색상 맞추는 반복 작업이 사라집니다.

⌨️ 발표 실전 단축키

발표 현장에서 마우스만 쓰면 프로답지 않아 보여요. 이 단축키들만 알면 발표가 훨씬 매끄러워져요.

단축키기능
F5처음 슬라이드부터 발표 시작
Shift+F5현재 슬라이드부터 발표 시작
B화면 블랙아웃 (잠시 가리기)
W화면 화이트아웃
Ctrl+P레이저 포인터 → 펜으로 전환
Esc발표 종료
숫자 + Enter해당 번호 슬라이드로 바로 이동
💡
B키 하나로 청중 집중도 올리기

발표 중 청중이 슬라이드가 아닌 나를 봐야 할 때 B키를 누르면 화면이 검게 되면서 청중의 시선이 발표자에게 집중돼요. 프로 발표자들이 자주 쓰는 테크닉이에요.

🤖 AI로 PPT 만들기

2026년에는 텍스트만 입력하면 AI가 슬라이드 전체를 만들어주는 도구들이 본격화됐어요.

4
감마 (Gamma) — 60초 만에 20장 슬라이드

내용만 입력하면 AI가 디자인·레이아웃·이미지까지 자동으로 만들어줘요. PPTX 다운로드도 가능합니다.

📌 gamma.app 접속 → 내용 입력 → AI 생성 → PPTX 다운로드
5
미리캔버스 AI PPT

한국어로 주제 입력 → AI가 한국어 슬라이드 자동 생성 → PPT 파일로 다운로드. 한국어 특화라 결과물 품질이 좋아요.

📌 miricanvas.com → AI 프레젠테이션 → 주제 입력 → 생성 → 다운로드
🤖
AI 생성 후 직접 다듬기 추천

AI PPT는 초안 생성에 최적이에요. 생성 후 슬라이드 마스터로 폰트·색상을 회사 표준에 맞게 다듬으면 완성도 높은 PT가 됩니다.

🎯 핵심 요약
  • 글꼴 일괄 변경: 홈 → 바꾸기(▼) → 글꼴 바꾸기 → 100장 3초 완료
  • 슬라이드 마스터: 보기 → 슬라이드 마스터 → .potx 저장 → 팀 공유
  • 발표 중 B키 = 화면 블랙아웃으로 청중 집중도 올리기
  • 무료 템플릿: 미리캔버스(한국어), 캔바(글로벌), 예스폼(비즈니스)
  • AI PPT 초안: 감마(gamma.app)로 60초 생성 → 마스터로 다듬기
