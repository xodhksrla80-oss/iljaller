❓ 한글 표에서 줄 추가가 안 되는 상황

한글에서 표 줄 추가가 안 될 때 원인은 대부분 두 가지입니다. 첫 번째는 추가 방법 자체를 모르는 경우이고, 두 번째는 표 자체가 잠겨 있는 경우예요.

ℹ️
한글 표 vs 엑셀 표 차이

엑셀처럼 셀을 클릭하고 드래그해서 행을 추가하는 방식이 아닙니다. 한글 표는 커서 위치와 마우스 우클릭 메뉴를 이용해서 줄을 추가합니다. 이 차이만 알면 바로 해결돼요.

⌨️ Tab키로 맨 끝 칸에서 줄 추가하기

가장 빠르고 쉬운 방법입니다. 표의 마지막 행, 마지막 칸에 커서를 놓고 Tab 키를 누르면 자동으로 새 행이 추가됩니다.

1
표 안에 커서 넣기

표 안의 아무 셀이나 클릭해서 커서를 위치시킵니다.

2
마지막 행, 마지막 칸으로 이동

Tab키를 계속 눌러서 맨 마지막 칸으로 이동하거나, 마우스로 마지막 칸을 직접 클릭합니다.

3
Tab 키 누르기 → 새 행 자동 추가

마지막 칸에서 Tab을 누르면 새 행이 아래에 자동으로 생깁니다. 계속 누르면 계속 추가돼요.

✅
Tab키 활용 팁

표 안에서 Tab을 누르면 다음 칸으로 이동합니다. 반대 방향은 Shift+Tab이에요. 단, 마지막 칸에서 Tab을 누를 때만 새 행이 추가됩니다. 가장 빠른 줄 추가 방법이에요.

🖱 마우스 우클릭으로 행·열 삽입하기

원하는 위치에 행을 중간에 삽입하거나, 여러 행을 한 번에 추가할 때는 마우스 우클릭 메뉴를 쓰면 됩니다.

1
행을 추가하고 싶은 위치의 셀에 커서 놓기

예: 3번째 행 아래에 새 행을 추가하고 싶다면 3번째 행의 아무 셀을 클릭합니다.

2
마우스 우클릭 → "줄/칸 추가/삭제" 선택

메뉴에서 "줄/칸 추가/삭제"를 클릭합니다. 한글 버전에 따라 메뉴 이름이 약간 다를 수 있어요.

3
추가 위치(위/아래) 선택 → 개수 입력 → 넣기

현재 행의 위 또는 아래를 선택하고, 추가할 행 수를 입력합니다. 한 번에 여러 줄도 가능해요.

💡
열(칸) 추가도 같은 방법

행 말고 열(칸)을 추가하고 싶을 때도 우클릭 → "줄/칸 추가/삭제"에서 동일하게 할 수 있어요. 왼쪽 또는 오른쪽 중 선택해서 추가하면 됩니다.

🔓 표 잠금 해제하는 방법

위 방법을 다 써봤는데도 줄 추가가 안 된다면 표가 잠겨 있는 것입니다. 공문서 양식이나 서식 파일에서 특히 많이 발생해요. 잠금을 해제해야 편집이 가능합니다.

1
표 안에 커서를 두고 → 표 선택 (F5 또는 표 테두리 클릭)

표 전체를 선택 상태로 만들어야 합니다. 표 테두리를 클릭하면 표 전체가 선택되면서 테두리에 파란 선이 나타나요.

2
마우스 우클릭 → "개체 속성" 선택

개체 속성 창이 열립니다.

3
"보호" 탭 → 체크 해제 → 설정

"개체 보호" 또는 "표 잠금" 관련 체크박스를 해제하고 확인을 클릭합니다. 이제 표 편집이 가능해집니다.

⚠️
문서 보호가 걸린 경우

문서 전체에 보호가 걸린 경우에는 개체 속성이 아닌 메뉴바 → 도구 → 문서 보호 설정에서 보호를 해제해야 합니다. 공문서 양식이라면 원본 제공처에 문의하는 게 나을 수 있어요.

🎯 핵심 요약
  • 가장 빠른 방법 → 표 마지막 칸에서 Tab키 누르기 → 새 행 자동 추가
  • 원하는 위치에 삽입 → 해당 행 우클릭 → "줄/칸 추가/삭제" 메뉴 이용
  • 표 안 이동 → Tab(다음 칸), Shift+Tab(이전 칸)
  • 위 방법으로 안 된다면 표가 잠긴 것 → 우클릭 → 개체 속성 → 보호 해제
  • 문서 전체 잠금은 도구 → 문서 보호 설정에서 해제
