📧 이메일 관리 꿀팁 6가지

📁 규칙으로 자동 분류
특정 발신자·키워드 이메일을 자동으로 지정 폴더로 이동. 팀장·클라이언트 메일만 별도 폴더로 자동 분리.
📌 홈 → 규칙 → 규칙 관리 → 새 규칙
⏰ 이메일 예약 발송
야근 중 작성한 메일을 다음 날 오전 9시에 발송되도록 예약. 상대방 입장에서 더 자연스럽게.
📌 메일 작성 → 보내기 옆 ▼ → 나중에 보내기
↩️ 보낸 메일 회수
보낸 메일을 상대방이 읽기 전 회수 가능. 같은 조직(Exchange) 내에서만 작동해요.
📌 보낸 편지함 → 메일 열기 → 작업 → 이 메시지 회수
📌 중요 메일 고정
처리해야 할 메일에 플래그를 달면 "플래그 표시된 항목"에서 한 번에 확인 가능. 할 일 목록처럼 활용.
📌 메일 우클릭 → 플래그 추가 → 알림 날짜 설정
🔕 집중 모드 (다른 편지함)
중요 메일만 "집중" 탭에, 나머지는 "기타" 탭에 자동 분류. 업무 집중력이 높아져요.
📌 보기 → 집중 받은 편지함 켜기
🔍 고급 검색
발신자·날짜·첨부파일 유무·키워드 조합으로 필터링. 수백 개 메일 중 원하는 것 바로 찾기.
📌 검색창 클릭 → 검색 도구 탭 → 조건 필터 활용

⚡ 업무 효율 꿀팁 5가지

⚡ 빠른 단계 (Quick Steps)
여러 작업을 버튼 하나로. "읽음 표시 + 특정 폴더 이동 + 전달"을 클릭 한 번에 실행.
📌 홈 → 빠른 단계 → 새 빠른 단계
📅 메일에서 바로 일정 추가
메일 내용을 캘린더 일정으로 바로 변환. 드래그해서 캘린더로 끌어다 놓으면 일정 자동 생성.
📌 메일을 캘린더 아이콘으로 드래그
🔄 서명 여러 개 관리
내부·외부·영문 서명을 별도로 만들어두고 메일마다 선택. 매번 복붙하는 번거로움 끝.
📌 파일 → 옵션 → 메일 → 서명
⌨️ 핵심 단축키
Ctrl+R 회신 / Ctrl+Shift+R 전체 회신 / Ctrl+F 전달 / Ctrl+1 받은편지함 / Ctrl+2 캘린더
🌐 번역 기능 내장
영문 이메일을 한국어로 바로 번역. 별도 번역기 불필요.
📌 메일 열기 → 번역 버튼 클릭 (또는 우클릭 → 번역)

🤖 Copilot AI 기능 4가지

Microsoft 365 구독자라면 아웃룩에 Copilot AI가 내장되어 있어요.

📋 이메일 스레드 요약
긴 이메일 체인을 AI가 3줄로 요약. 100통 스레드도 핵심만 빠르게 파악.
📌 메일 상단 Copilot 버튼 → 이 대화 요약
✍️ 답장 초안 자동 작성
메일 내용 기반으로 AI가 답장 초안 작성. 톤(공식적/친근한) 선택 가능.
📌 회신 클릭 → Copilot → 초안 작성
📊 회의 준비 요약
캘린더 일정 클릭 시 관련 이메일·파일을 Copilot이 자동 수집·요약. 회의 전 5분 브리핑.
📌 캘린더 일정 → Copilot 탭
🔍 자연어 이메일 검색
"지난주 김대리가 보낸 계약서 관련 메일 찾아줘"처럼 자연어로 검색 가능.
📌 검색창 → Copilot으로 검색
💡
Copilot은 Microsoft 365 Business 이상에서 사용 가능

Copilot for Microsoft 365는 기업 구독 플랜에 포함돼요. 개인 Microsoft 365 구독자는 일부 기능만 사용 가능합니다.

🎯 핵심 요약
  • 규칙 자동 분류 — 클라이언트·팀장 메일 자동으로 별도 폴더 이동
  • 예약 발송 — 야근 중 작성 후 다음 날 아침 자동 발송
  • 빠른 단계 — 반복 작업 버튼 하나로 묶기
  • Ctrl+R 회신 / Ctrl+1 받은편지함 — 단축키로 속도 2배
  • Copilot AI — 이메일 3줄 요약·답장 초안·자연어 검색
