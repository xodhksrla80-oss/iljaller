📋 화면 캡처 방법, 이렇게 많았어?

일하다 보면 진짜 화면 캡처 쓸 일이 많잖아요? 그런데 대부분은 Print Screen이나, 가끔은 폰으로 모니터를 그냥 찍는 경우도 많더라고요. 사실 윈도우, 맥 모두 생각보다 훨씬 편리한 내장 캡처 기능이 있거든요! 이거만 알아도 일 처리 속도가 달라집니다.

8가지
윈도우 + 맥 화면 캡처 단축키 완전 정리
💡
이젠 폰으로 모니터 찍지 마세요!

솔직히 폰으로 찍으면 화질 울렁~ 깨져서 문서에 넣기도 별로예요. 단축키 한 번만 쓰면 선명하게 바로 캡처하고 바로바로 붙여넣을 수 있답니다.

🪟 윈도우 화면 캡처 단축키 완전 정리

윈도우에서 제일 자주 쓰는 단축키 4개만 콕 집어서 소개해요. 상황 보고 골라 쓰면 됩니다!

Win + Shift + S
부분 캡처 (가장 추천!)
원하는 영역 드래그 → 자동 클립보드 저장 → 붙여넣기 완. 전. 가능! 윈도우 10/11이면 다 돼요.
Win + PrtSc
전체 화면 자동 저장
전체 화면 한 번에 캡처, 파일로 바로 저장돼서 폴더(사진 → 스크린샷)에 저장돼요.
PrtSc
전체 화면 클립보드 복사
전체 화면을 클립보드에만 복사. Ctrl+V로 붙여넣어야 사용 가능. 파일로 저장되지 않음.
Alt + PrtSc
현재 창만 캡처
현재 활성화된 창만 캡처해서 클립보드에 복사. 여러 창이 겹쳐 있을 때 유용.
⌨️
Win + Shift + S 를 무조건 외우세요

특정 부분만 뽑아서 캡처하고, 바로 클립보드로 복사돼서 카톡, 메일, 워드, 뭐든 즉시 붙여넣기 가능! 저는 하루에도 여러 번 씁니다.

🔧 윈도우 캡처 도구 활용법

윈도우 11에는 '캡처 도구(Snipping Tool)'라는 내장 앱도 있어요. 이 앱 쓰면 단순 캡처뿐 아니라 영상 녹화, 텍스트 인식, 간단 편집까지 가능해서 더 똑똑하게 활용할 수 있답니다.

열기
Win + Shift + S 또는 검색창에 "캡처 도구" 입력

캡처 도구 앱을 열면 캡처 모드를 선택할 수 있어요. 사각형, 창, 전체 화면, 자유 형식 4가지 모드 지원.

녹화
화면 녹화 기능 (윈도우 11 전용)

캡처 도구 앱에서 동영상 아이콘(🎥)을 선택하면 화면 녹화도 됩니다. 간단한 튜토리얼이나 업무 설명 영상 만들 때 유용해요.

OCR
텍스트 추출 기능 (PowerToys 활용)

마이크로소프트 PowerToys 앱을 설치하면 화면의 텍스트를 드래그해서 복사하는 "텍스트 추출기" 기능이 생깁니다. PDF나 이미지의 텍스트도 바로 복사 가능.

게임
게임바 캡처: Win + Alt + PrtSc

게임 중에 캡처할 때는 Xbox Game Bar 단축키를 사용하세요. 게임 성능에 영향 없이 빠르게 캡처됩니다. 저장 위치: 동영상 → 캡처 폴더.

🍎 맥(Mac) 화면 캡처 단축키 완전 정리

맥은 윈도우보다 캡처 기능이 훨씬 더 직관적이고 강력합니다. 단축키 3가지만 외우면 모든 상황을 커버할 수 있어요.

기능단축키특징
전체 화면 캡처Command+Shift+3바탕화면에 PNG 파일로 저장
부분 캡처Command+Shift+4드래그로 영역 선택 후 저장
창 단위 캡처Command+Shift+4 → Space창 위에 마우스 올리고 클릭
스크린샷 도구Command+Shift+5캡처+녹화 모드 선택 가능
클립보드 복사위 단축키 + Control파일 저장 없이 붙여넣기만
🍎
맥 캡처 저장 위치 변경하기

Command+Shift+5를 눌러보면 아래쪽에 '옵션' 메뉴가 보여요. 거기서 원하는 폴더로 저장 위치를 변경할 수 있답니다. 기본값이 바탕화면이라 파일이 막 쌓이기 쉬운데, 자주 쓰는 폴더로 바꿔두면 정리할 때 한결 편해지더라고요!

📁 캡처 이미지 저장 위치와 활용 팁

저처럼 캡처를 자주 하다 보면 파일이 여기저기 흩어지기 일쑤죠. 어디에 저장되는지만 잘 알아도 찾기 훨씬 쉬워지고, 깔끔하게 관리할 수 있어요.

📂
윈도우 자동 저장 위치: 사진 → 스크린샷 폴더

Win+PrtSc로 캡처하면 C:\사용자\이름\사진\스크린샷 폴더에 자동 저장돼요. 파일명은 '스크린샷(날짜).png'처럼 저장되니까 정리하기도 쉽죠.

📋
클립보드 복사 활용: 바로 문서·메신저에 붙여넣기

Win+Shift+S로 캡처하면 이미지가 바로 클립보드에 들어가요. 따로 파일 저장 안 해도, 카카오톡, 이메일, 워드, 엑셀 어디든 Ctrl+V만 눌러서 바로 붙여넣을 수 있어요. 시간 절약도 되고 진짜 편해요!

✏️
간단 편집: 캡처 후 바로 나오는 편집 창 활용

Win+Shift+S로 캡처하면 화면 오른쪽 아래에 작은 미리보기가 잠깐 떠요! 그걸 누르면 편집 창이 바로 열리거든요. 화살표, 텍스트, 형광펜 등 표시를 바로 추가할 수 있으니 따로 그림판 열 필요가 없어요.

🖼
파일 형식: PNG vs JPG 차이

기본 저장 형식은 PNG입니다. 텍스트가 있는 화면은 PNG가 더 선명해요. 사진이나 용량을 줄일 때는 JPG가 유리합니다. 그림판에서 열어서 "다른 이름으로 저장"하면 형식 변환 가능.

🎯
카톡 전송, 이 조합이 꿀팁!

Win+Shift+S로 필요한 부분만 캡처하고 → 카카오톡 채팅창 클릭 → Ctrl+V만 눌러주면 끝! 파일로 저장 안 해도 되고, 화면 내용을 바로바로 전송할 수 있어서 진짜 업무 속도가 빨라져요😆

🎯 핵심 요약
  • 윈도우 부분 캡처: Win + Shift + S (가장 자주 쓰는 필수 단축키)
  • 윈도우 전체 저장: Win + PrtSc → 사진\스크린샷 폴더에 자동 저장
  • 윈도우 현재 창만: Alt + PrtSc
  • 맥 전체 캡처: Command+Shift+3 / 부분 캡처: Command+Shift+4
  • 맥 캡처 도구 (녹화 포함): Command+Shift+5
  • 캡처 후 Ctrl+V 붙여넣기 → 카카오톡·문서에 즉시 활용 가능
