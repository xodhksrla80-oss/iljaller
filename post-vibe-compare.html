⚡ 세 줄 요약 — 어떤 툴이 내 것?

고민되시죠? 일단 이것만 보고 결정하세요.

⌨️
Cursor

코드를 직접 보면서 작업하고 싶은 분. 진짜 서비스까지 만들 생각이라면 이걸로.

💜
Lovable

코딩 하나도 몰라도 괜찮아요. 아이디어만 있으면 대화하듯 앱을 만들 수 있어요.

⚡
Bolt

일단 빠르게 프로토타입부터. "이게 가능한지" 테스트해보고 싶을 때 딱이에요.

숫자로 보면 더 명확해요.

Bolt가 가장 저렴하지만 복잡한 앱은 한계가 있어요. Lovable은 비싸지만 코딩 없이 가장 완성도 높은 결과물이 나옵니다. Cursor는 중간 가격에 확장성 최고.

Cursor
cursor.com · AI 코드 편집기의 표준
$20/월
무료 플랜 있음

VS Code 기반 AI 편집기예요. 코드를 직접 보면서 AI에게 수정·추가를 요청할 수 있어요. 2025년 개발자들 사이에서 가장 인기 있는 바이브코딩 툴입니다.

  • 🔵 복잡한 앱도 완성 가능
  • 🔵 코드 직접 수정 가능
  • 🔵 GitHub 연동 완벽
  • 🔵 무료 플랜으로 시작 가능
⚠️ 단점
  • 🔴 코딩 기초 있어야 편함
  • 🔴 처음엔 설정이 복잡
  • 🔴 에러 해결에 시간 필요
💜
Lovable
lovable.dev · 대화만으로 앱을 만든다
$39/월
무료 체험 가능

브라우저에서 한국어로 "이런 앱 만들어줘"라고 입력하면 30초 안에 실제 동작하는 앱이 만들어져요. 코딩 지식이 전혀 없어도 됩니다. 비싼 게 단점이지만 결과물 품질은 훌륭해요.

✅ 장점
  • 🟣 코딩 0% 진입 가능
  • 🟣 30초 안에 앱 완성
  • 🟣 한국어 입력 지원
  • 🟣 결과물 품질 높음
⚠️ 단점
  • 🔴 셋 중 가장 비쌈
  • 🔴 복잡한 커스터마이징 어려움
  • 🔴 규모 커지면 한계
⚡
Bolt
bolt.new · 설치 없이 브라우저에서 바로
$15/월
무료 플랜 있음

아무것도 설치 안 해도 됩니다. bolt.new 접속해서 원하는 것만 입력하면 끝이에요. "이런 기능이 실현 가능한지" 빠르게 확인하고 싶을 때 가장 편리합니다.

✅ 장점
  • 🟢 설치 필요 없음
  • 🟢 셋 중 가장 저렴
  • 🟢 프로토타입 초고속
  • 🟢 무료로도 충분히 테스트
⚠️ 단점
  • 🔴 복잡해지면 한계 명확
  • 🔴 프로덕션 수준엔 부족
  • 🔴 커스터마이징 제한

🎯 수준별 추천

내 상황에 맞는 걸 골라보세요.

"코딩 좀 해봤고, 진짜 서비스까지 만들고 싶다"

풀스택 개발이 목표라면 처음부터 Cursor. 러닝커브가 있지만 확장성이 압도적입니다.

⌨️ Cursor
"코딩 하나도 모르는데, 아이디어는 있어"

Lovable이 진입장벽이 가장 낮아요. 한국어로 설명하면 AI가 다 만들어줍니다.

💜 Lovable
"일단 어떻게 생겼는지만 빠르게 보고 싶어"

bolt.new 접속해서 바로 시작. 설치 없이 3분 안에 결과물 확인 가능합니다.

⚡ Bolt
"돈 아끼고 싶은데 어디서 시작해야 할지 모르겠어"

Bolt 무료로 먼저 감 잡고 → 익숙해지면 Cursor 무료 플랜 이동하는 게 정석 루트예요.

⚡ Bolt 먼저

🔑
결국 핵심은 '프롬프트'예요

어떤 툴을 써도 지시를 잘할수록 결과가 달라집니다. "버튼 만들어줘"보다 "오른쪽 하단에 파란색 둥근 버튼, 클릭하면 장바구니에 추가되게 해줘"처럼 구체적으로 쓸수록 원하는 결과가 나와요.

🎯 핵심 요약
  • 코딩 경험 있고 진짜 서비스 만들 거라면 → Cursor ($20/월)
  • 코딩 몰라도 아이디어를 앱으로 만들고 싶다면 → Lovable ($39/월)
  • 일단 빠르게 테스트만 해보고 싶다면 → Bolt ($15/월, 무료 가능)
  • 현장 추천 조합 → Lovable로 프로토타입 → Cursor로 완성
  • 툴보다 중요한 건 프롬프트. 구체적으로 지시할수록 결과가 달라진다
