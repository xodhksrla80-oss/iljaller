⌨️ 노션 AI 호출하는 법

노션 AI는 별도 앱이 아니에요. 노션 페이지 안에서 스페이스바를 누르면 AI 메뉴가 바로 뜹니다. 글 쓰다가 막히거나, 기존 내용을 수정하고 싶을 때 언제든 호출 가능해요.

⌨️
노션 AI 호출 2가지 방법

① 빈 줄에서 스페이스바 → AI 글쓰기 메뉴 등장
② 텍스트 선택 후 Ask AI 클릭 → 선택 내용 기반으로 요약·수정·번역

🚀 직장인이 매일 쓰는 기능 6가지

1
✍️ 문서 초안 자동 작성
주제만 입력하면 AI가 전체 문서 구조와 내용을 잡아줘요. 기획서·제안서·보고서 첫 페이지 막막할 때 최고.
📌 스페이스바 → "초안 작성" → 주제 입력
2
📋 회의록·긴 문서 요약
긴 텍스트를 선택하면 AI가 핵심만 3~5줄로 요약해줘요. 클로바노트로 추출한 회의 텍스트를 붙여넣고 요약 요청하면 5초 만에 회의록 완성.
📌 텍스트 선택 → Ask AI → "요약해줘"
3
🌐 번역 (한↔영 즉시)
영문 메일·계약서를 붙여넣고 번역 요청하면 바로 한국어로. 반대로 한국어 문서를 영문으로 번역도 OK.
📌 텍스트 선택 → Ask AI → "한국어로 번역"
4
✅ 맞춤법·어조 교정
작성한 문서의 맞춤법 교정 + 어조를 "공식적으로" 또는 "친근하게" 바꿔줘요. 이메일·공문서 다듬기에 딱.
📌 텍스트 선택 → Ask AI → "더 공식적인 톤으로 수정"
5
🔍 내용 확장 (Expand)
짧게 쓴 아이디어나 개요를 AI가 살을 붙여 확장해줘요. 제안서 항목별로 한 줄씩 쓴 뒤 확장 요청하면 빠르게 완성.
📌 스페이스바 → "계속 쓰기" 또는 선택 → "더 자세히"
6
❓ Q&A — 페이지 내 질문
페이지 안에서 "이 문서의 핵심 3가지가 뭐야?"라고 질문하면 AI가 해당 페이지 내용을 기반으로 답변해줘요.
📌 스페이스바 → "질문하기" → 궁금한 점 입력

💰 요금제 — 무료로도 쓸 수 있나

💰
노션 AI는 별도 추가 비용

노션 무료 플랜 사용자도 AI를 제한적으로 체험 가능(약 20회). 정식으로 쓰려면 AI 애드온 월 $10(연간 결제 시 $8)을 추가해야 해요. 노션 플러스 이상 구독자는 AI 포함 플랜 선택 가능.

🔄 실전 루틴 — 회의록 자동화

✅
5분 만에 회의록 완성하는 루틴

① 클로바노트로 회의 녹음 → 텍스트 추출
② 노션 새 페이지에 텍스트 붙여넣기
③ 전체 선택 → Ask AI → "회의록 형식으로 요약해줘 (핵심 결정사항, 액션 아이템 포함)"
④ 완성된 회의록 팀원과 공유

총 소요 시간: 5분 이내

🎯 핵심 요약
  • 스페이스바 → AI 메뉴 호출. 텍스트 선택 → Ask AI로 즉시 수정·요약·번역
  • 회의록: 클로바노트 텍스트 → 노션 붙여넣기 → AI 요약 = 5분 완성
  • 문서 초안: 주제 입력 → AI 초안 → 사람이 수정 = 2배 빠른 작성
