💬 메시지 관련 꿀팁 5가지

1
⏰ 메시지 예약 전송

늦은 밤 메시지를 보내야 할 때, 아침에 보내야 할 때 미리 예약해두면 돼요. 개인톡과 단톡방 모두 가능합니다. (오픈채팅은 불가)

📌 방법: 채팅창 → 메시지 입력 → 전송 버튼 길게 누르기 → 예약 전송 → 날짜·시간 설정
2
✏️ 보낸 메시지 수정

잘못 보낸 메시지를 삭제만 할 수 있는 게 아니에요. 전송 후 5분 이내라면 수정도 가능해요. 상대방이 이미 읽었어도 수정 가능합니다.

📌 방법: 보낸 메시지 길게 누르기 → 수정 → 내용 변경 후 완료
3
📌 메시지 고정 (핀)

중요한 메시지를 채팅방 상단에 고정할 수 있어요. 중요한 공지사항, 약속 장소, 계좌번호 등을 고정해두면 매번 스크롤할 필요가 없어요.

📌 방법: 고정할 메시지 길게 누르기 → 고정 → 채팅방 상단에 표시됨
4
🔗 링크 미리보기 끄기

링크 보낼 때 자동으로 생기는 미리보기가 거슬릴 때 있잖아요. 링크 앞뒤에 공백을 넣거나 링크 주소 일부를 괄호로 감싸면 미리보기 없이 전송돼요.

📌 방법: 링크 앞에 공백 한 칸 추가 → 미리보기 없이 전송
5
📋 나에게 메시지 보내기

급하게 메모할 때, 다른 기기로 파일 전송할 때 "나"에게 메시지 보내면 일종의 클라우드 메모장처럼 써요. 직장인들 엄청 많이 씁니다.

📌 방법: 친구 탭 → 내 프로필 클릭 → 나에게 보내기

👤 프로필·프라이버시 꿀팁 4가지

6
👥 멀티프로필 — 직장용·친구용 분리

하나의 계정으로 최대 4개(기본 1개 + 추가 3개)의 프로필을 만들어 특정 친구에게만 다른 프로필을 보여줄 수 있어요. 직장 상사에겐 업무용, 친구에겐 개인용으로 분리하면 딱이에요.

📌 방법: 친구 탭 → 우측 상단 톱니바퀴 → 멀티프로필 만들기 → 적용할 친구 선택
7
🔒 프로필 업데이트 나만 보기

프로필 사진 바꿀 때마다 친구 피드에 알림이 가는 게 싫다면 이 설정을 켜두세요. 조용히 프로필 변경할 수 있어요.

📌 방법: 내 프로필 → 설정 → 프라이버시 보호 설정 → 프로필 업데이트를 나만 보기
8
👁️ 친구에게만 게시물 공개

카카오톡 피드에 올린 게시물을 친구에게만 보이도록 설정할 수 있어요. 모르는 사람이 추천 친구로 뜰 때 내 프로필이 노출되는 게 걱정된다면 꼭 설정하세요.

📌 방법: 내 프로필 → 설정 → 프라이버시 보호 설정 → 친구에게만 게시물 공개 ON
9
🚫 특정 친구에게 내 프로필 숨기기

멀티프로필을 사용하면 특정 친구에게만 다른 프로필을 보여주거나, 아예 숨길 수도 있어요. 불편한 관계를 정리하기도 편해요.

📌 방법: 멀티프로필 → 친구 관리 → 지정 친구 설정에서 숨기고 싶은 친구 제외

🔍 채팅방 활용 꿀팁 3가지

10
🔍 채팅방 내 메시지 검색

오래된 대화에서 특정 내용 찾을 때 위로 무한 스크롤 하시나요? 채팅방 내 검색 기능을 쓰면 바로 찾을 수 있어요. 계좌번호, 주소, 약속 내용 찾을 때 최고예요.

📌 방법: 채팅방 우측 상단 ≡ → 검색 아이콘 → 키워드 입력
11
🗂️ 톡서랍 — 사진·파일 자동 백업

카카오톡으로 받은 사진과 파일이 자동으로 클라우드에 백업돼요. 폰 바꿔도 대화 내용과 파일이 살아있어요. 무료 용량은 1GB, 유료는 더 넓게 쓸 수 있어요.

📌 방법: 더보기 탭 → 서랍 → 설정에서 자동 백업 ON
12
🔔 특정 채팅방만 알림 끄기

단톡방 알림이 너무 많을 때 전체 알림을 끄면 중요한 연락을 놓쳐요. 특정 채팅방만 골라서 알림을 끄는 게 훨씬 똑똑해요.

📌 방법: 채팅방 우측 상단 ≡ → 알림 → 알림 끄기 (1시간/8시간/24시간/계속)

⚡ 편의 기능 꿀팁 3가지

13
💸 카카오페이 송금 — 계좌 없이

계좌번호 몰라도 카카오톡 친구에게 바로 송금 가능해요. 더치페이할 때 정산 요청 기능도 있어서 각자 낼 금액을 자동 계산해줘요.

📌 방법: 채팅방 → + 버튼 → 송금 or 정산
14
📍 현재 위치 실시간 공유

약속 장소 찾기 어려울 때 내 현재 위치를 실시간으로 공유할 수 있어요. 상대방이 지도에서 내 위치를 실시간으로 확인할 수 있어서 길 찾기 훨씬 편해요.

📌 방법: 채팅방 → + 버튼 → 위치 → 현재 위치 or 실시간 위치 공유
15
🎵 무음 이모티콘 전송

이모티콘 보낼 때 상대방한테 소리 나는 게 미안할 때 있잖아요. 이모티콘을 길게 눌러서 보내면 소리 없이 전송돼요. 야근 중인 상대방 배려 완료.

📌 방법: 이모티콘 선택 → 길게 누르기 → 무음 전송 선택
💡
더블 터치 조심하세요!

친구 탭에서 프로필 사진을 더블 터치하면 자동으로 좋아요가 눌려요. 상대방에게 알림이 가니 친하지 않은 분 프로필에서는 주의하세요!

🎯 핵심 요약
  • 예약 전송: 전송 버튼 길게 누르기 → 야근 중 새벽 메시지 예약
  • 멀티프로필: 최대 4개 → 직장용/친구용 완전 분리 가능
  • 채팅방 내 검색: 우측 상단 ≡ → 오래된 메시지 바로 찾기
  • 프로필 업데이트 나만 보기: 조용히 프로필 바꾸기 가능
  • 나에게 메시지 보내기: 메모장·파일 전송용 클라우드로 활용
