📑 탭 관련 꿀팁 (1~4번)

크롬 꿀팁, 한 번만 알아두면 쭉 써먹게 되니까 꼭 읽어보세요! 저도 평소에 자주 쓰는 기능들이고요, 몇 가지는 몰랐다면 정말 유용하게 느끼실 거예요.😄

1
실수로 닫은 탭 복구

급하게 탭 닫았다가 멘붕 올 때 있잖아요? 이럴 땐 이 단축키만 한 구세주가 없더라고요. 누르면 마지막에 닫았던 탭이 그 상태 그대로 복구돼요. 여러 번 누르면 그 전 탭들까지 차례로 다 돌아옵니다!

Ctrl+Shift+T
2
탭 그룹으로 업무 정리

탭이 너무 많아져서 뭘 보고 있었는지 헷갈린 적 있으시죠? 탭을 마우스 오른쪽 버튼으로 클릭해서 '새 그룹에 탭 추가'만 눌러보세요. 색상 정할 수도 있고, '기획 자료', '참고', '메일' 등 이름도 맘대로 붙일 수 있어요.

탭 우클릭 → 새 그룹
3
탭 번호로 이동

Ctrl 키랑 숫자키(1~8) 조합만 기억해두세요. 탭이 많을 때 마우스 찾느라 시간 버리는 경우 많은데, 이 단축키만 알면 손쉽게 원하는 탭으로 이동! Ctrl+9 누르면 언제나 맨 마지막 탭으로 갈 수 있어요.

Ctrl+1~9
4
탭 고정(핀)하기

메일이나 캘린더처럼 항상 떠 있어야 하는 사이트, 그럴 땐 탭을 우클릭해서 '탭 고정'만 선택하면 끝! 자동으로 왼쪽에 작게 자리 잡고 실수로 닫혀버릴 일도 없어요.

탭 우클릭 → 탭 고정

🔍 주소창 제대로 활용하기 (5~8번)

5
주소창이 계산기?!

Ctrl+L로 주소창에 바로 가서 숫자 계산식만 입력해도 바로 결과가 딱! "500 * 1.1"처럼 부가세 계산, "180cm in feet"처럼 단위 변환까지 다 됩니다. 한 번 써보면 습관돼요.

Ctrl+L → 수식 입력
6
사이트 내 검색 단축키

유튜브, 나무위키, 쿠팡처럼 자주 가는 사이트 이름을 주소창에 치고, 바로 Tab 키를 눌러보세요. 그러면 그 사이트 안에서 바로 검색할 수 있는 모드가 됩니다. 생각보다 엄청 편해요!

사이트명 입력 후 Tab
7
주소창 바로 포커스

Ctrl+L 또는 F6만 누르면 주소창으로 바로 이동하고, URL 전체가 선택되어 있어요. 바로 새 주소를 입력하거나, 기존 주소 복사할 때 진짜 유용합니다. 마우스 필요 없어요!

Ctrl+L 또는 F6
8
시크릿 모드 열기

Ctrl+Shift+N만 기억하면 어디서든 시크릿 창이 열립니다. 방문 기록이나 쿠키 남기고 싶지 않을 때, 여러 명이 같이 쓰는 PC 쓸 때, 항공권·호텔처럼 가격 자주 바뀌는 사이트 보기에도 최고예요!

Ctrl+Shift+N
💡
주소창 계산기 활용 예시

"500 * 1.1" → 부가세 포함 금액 / "180cm in feet" → 키 단위 변환 / "1달러 원화" → 실시간 환율 / "2026-01-01에서 오늘까지 며칠" → 날짜 계산

🔌 직장인 추천 확장 프로그램 (9~10번)

저처럼 한 번에 탭을 20개, 30개씩 켜놓는 분들 많죠? 확장 프로그램 단축키를 설정하면 더 빠르게 씁니다. 주소창에 chrome://extensions/shortcuts을 입력하면 나만의 단축키 지정 페이지가 바로 열려요!

확장 프로그램 핵심 기능 추천 이유
📌 OneTab 탭 전부 목록으로 저장 클릭 한 번에 모든 탭이 리스트로 모아져 브라우저 가벼워짐. 필요할 때만 다시 열 수 있어요
🖼️ GoFullPage 페이지 전체 스크린샷 길~게 스크롤하는 자료도 한 장짜리 이미지로 저장. 보고서 첨부에 아주 유용, 무료!
📝 Grammarly 영문 문법 교정 영어 메일·문서 작성 시 자동으로 오류 표시
🎵 Video Speed Controller 영상 재생 속도 조절 유튜브·강의를 2~4배속으로 재생. 학습 시간 절반
✅
확장 프로그램 단축키 설정법

chrome://extensions/shortcuts 접속 → 사용하고 싶은 확장 프로그램 옆 입력란 클릭 → 원하는 키 조합 입력. 예: Ctrl+Shift+1로 OneTab 활성화.

⚙️ 숨겨진 설정들도 한번 써보세요

크롬에 내장된 관리 페이지들은 북마크해두면 정말 편리합니다. 원하는 곳으로 곧장 이동 가능해서 설정 찾아 헤매는 시간이 확 줄어요!

🔧
크롬 내장 페이지 — 북마크해두면 편리

chrome://settings → 전체 설정 / chrome://extensions → 확장 프로그램 관리 / chrome://flags → 실험적 기능(고급 사용자용) / chrome://downloads → 다운로드 기록 / chrome://history → 방문 기록

🔒
쿠키 팝업 귀찮으셨죠? 이거 설치하면 신세계!

I don't care about cookies 확장 프로그램을 설치하면 대부분의 사이트에서 쿠키 동의 팝업이 자동으로 꺼집니다. 저도 이거 없으면 못 써요😆

단축키 기능 언제 쓰면 좋을까?
Ctrl+Shift+T 닫은 탭 복구 실수로 탭 닫았을 때 즉시
Ctrl+1~9 탭 번호 이동 탭 많을 때 마우스 없이 이동
Ctrl+L 주소창 포커스 + 계산기 URL 이동, 계산, 환율 변환
Ctrl+Shift+N 시크릿 모드 공용 PC, 계정 전환, 가격 비교
사이트명+Tab 사이트 내 검색 유튜브, 쿠팡, 나무위키 검색
탭 우클릭 탭 그룹·고정 프로젝트별 탭 정리, 필수탭 고정
🎯 핵심 요약 — 오늘 바로 써보기
  • 닫은 탭 복구: Ctrl+Shift+T — 여러 번 누르면 그 전 탭들까지 차례로!
  • 탭 그룹: 우클릭 → '새 그룹에 탭 추가' — 색상+이름으로 프로젝트별 정리
  • 탭 번호 이동: Ctrl+1~9 — 마우스 없이 탭 전환, Ctrl+9는 마지막 탭
  • 주소창 계산기: Ctrl+L 후 수식 입력 — 부가세·환율·단위변환 다 됨
  • 시크릿 모드: Ctrl+Shift+N — 공용PC, 계정 전환, 가격 비교 시 활용
  • 확장 단축키 설정: chrome://extensions/shortcuts 에서 나만의 단축키
  • 필수 확장: OneTab(탭 정리) + GoFullPage(전체 캡처) — 둘 다 무료!
