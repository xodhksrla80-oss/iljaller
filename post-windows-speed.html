🔍 느려진 원인 먼저 파악하기

먼저, 컴퓨터가 왜 느려졌는지부터 알아봐야겠죠? 최적화를 시작하기 전에 원인을 정확히 파악하면, 쓸데없는 고생을 줄일 수 있거든요. Ctrl + Shift + Esc를 눌러 작업 관리자를 열어보세요. 여기서 CPU, 메모리, 디스크 사용률을 실시간으로 볼 수 있어요. CPU가 90%를 넘으면 뭔가 프로그램이 엄청 잡아먹고 있는 거고, 메모리가 90% 이상이면 램이 부족하다는 신호예요. 디스크 사용률이 100%라면... 이제 슬슬 SSD로 바꿔야 할 타이밍일 수도 있습니다! 이렇게 원인을 먼저 찾아야 진짜 해결이 쉬워져요.

📊
작업 관리자로 원인 한눈에 파악

CPU 90% 이상 → 특정 앱이 점유 중 (프로세스 탭에서 범인 찾기) / 메모리 90% 이상 → RAM 부족, 탭·앱 줄이기 / 디스크 100% → SSD 교체 고려 또는 임시파일 정리

항목 위험 수준 대처법
CPU 90% 이상 프로세스 탭에서 점유 앱 종료
메모리 90% 이상 탭·앱 줄이기, RAM 업그레이드 고려
디스크 100% 임시파일 정리 또는 SSD 교체

🚀 방법 1~3 — 바로 효과 볼 수 있는 것들

1
시작 프로그램 줄이기

컴퓨터 켜자마자 자동으로 뜨는 프로그램이 많을수록 부팅 느려지는 건 당연하겠죠? 설정 → 앱 → 시작 프로그램으로 들어가서 카카오톡, Zoom, OneDrive, Discord 이런 거, 꼭 부팅할 때 필요하지 않은 애들은 꺼주세요. 실제로 부팅 속도가 훨씬 빨라집니다, 완전 체감돼요😊

설정 → 앱 → 시작 프로그램
2
임시 파일·디스크 정리

윈도우를 쓰다 보면 임시 파일이 쑥쑥 쌓여요. 시작 메뉴에서 '디스크 정리'라고 검색한 다음, C드라이브 고르고, '시스템 파일 정리'까지 눌러주면 몇 GB는 바로 확보할 수 있을 거예요. 6개월에 한 번만 해줘도 컴퓨터가 한결 가벼워지더라고요!

시작 → 디스크 정리 → C드라이브
3
백그라운드 앱 끄기

쓰지도 않는 앱이 계속 켜져 있으면 CPU랑 메모리만 괜히 잡아먹어요. 설정 → 앱 → 설치된 앱에서 하나씩 골라서, 백그라운드 권한을 '안 함'으로 바꿔주면 속도가 훨씬 나아집니다.

설정 → 앱 → 설치된 앱 → 백그라운드 앱 권한

⚙️ 방법 4~5 — 조금 더 깊이 들어가기

4
전원 설정 변경

제어판에서 '전원 옵션' 들어가면 기본이 '균형' 모드인데, 작업할 땐 과감하게 '고성능'으로 선택! 확실히 속도가 달라진 느낌 받을 수 있어요. 참고로 노트북은 충전기 꽂았을 때만 '고성능'으로 두는 게 배터리에도 부담이 덜 갑니다. 이거 은근 꿀팁이죠?

제어판 → 전원 옵션 → 고성능
5
C드라이브 여유 공간 15% 이상 유지

C드라이브가 꽉 차면 윈도우 업데이트 실패, 가상 메모리 부족으로 속도가 급격히 떨어집니다. 512GB 드라이브라면 최소 77GB, 즉 전체 용량의 15% 이상은 항상 비워두세요. 대용량 영상, 사진 파일은 D드라이브나 외장하드로 옮겨두는 게 좋습니다.

드라이브 용량 최소 여유 공간 (15%) 이하 시 증상
256GB 38GB 이상 업데이트 실패, 속도 저하
512GB 77GB 이상 가상 메모리 부족
1TB 153GB 이상 느린 파일 작업
💡
윈도우11 '효율성 모드' 활용

작업 관리자에서 CPU를 많이 쓰는 프로세스를 우클릭 → '효율성 모드' 설정하면 해당 앱의 우선순위를 낮춰 다른 작업에 여유를 줍니다. 백그라운드 앱에 유용합니다.

🔧 방법 6~7 — 심화 최적화

6
시각 효과 줄이기

윈도우의 애니메이션과 투명 효과는 GPU 자원을 사용합니다. 내 PC를 우클릭 → 속성 → 고급 시스템 설정 → 성능 설정에서 '최적 성능으로 조정'을 선택하면 시각 효과가 줄어들고 체감 속도가 올라갑니다. 구형 PC일수록 효과가 큽니다.

내 PC 우클릭 → 속성 → 고급 시스템 설정 → 성능
7
윈도우 업데이트 최신 유지

업데이트를 미루다 보면 성능 버그가 쌓입니다. 특히 드라이버 업데이트 누락은 속도 저하로 이어지는 경우가 많습니다. 설정 → Windows 업데이트에서 정기적으로 최신 상태를 유지하고, 업데이트 완료 후 재시작까지 해줘야 완전히 적용됩니다.

설정 → Windows 업데이트 → 업데이트 확인

🔩 그래도 느리다면? 하드웨어 점검

위 방법을 모두 적용해도 느리다면 소프트웨어 한계가 아닌 하드웨어 문제일 가능성이 높습니다. RAM이 8GB 이하인데 크롬 탭을 10개 이상 열면 느린 게 당연합니다. 16GB로 업그레이드하면 체감이 확실히 달라집니다. HDD를 쓰고 있다면 SSD 교체가 가장 드라마틱한 속도 개선 효과를 냅니다.

⚠️
하드웨어 업그레이드 판단 기준

RAM이 8GB 이하인데 크롬 탭 10개 이상을 동시에 열면 느린 게 당연합니다. 16GB로 업그레이드하면 체감이 완전히 달라집니다. HDD 사용 중이라면 SSD 교체가 가장 드라마틱한 효과를 냅니다.

✅
비용 대비 효과 순위

① HDD → SSD 교체 (가장 극적인 체감) → ② RAM 8GB → 16GB 업그레이드 → ③ CPU 업그레이드 (보통 마더보드 교체까지 필요해서 비용 큼)

🎯 핵심 요약
  • 작업 관리자로 원인 파악 → Ctrl+Shift+Esc로 CPU·메모리·디스크 확인
  • 시작 프로그램 줄이기 → 설정 → 앱 → 시작 프로그램, 불필요한 것 끄기
  • 디스크 정리 → 6개월에 한 번만 해도 컴퓨터가 한결 가벼워져요
  • 전원 옵션 고성능 설정 → 충전기 연결 시 특히 효과적
  • C드라이브 15% 여유 공간 유지 → 꽉 차면 속도 급저하
  • 그래도 느리면 → RAM 16GB · SSD 교체가 가장 드라마틱한 해결책
