🎨 색상으로 일정 분류하기 — 한눈에 파악

구글 캘린더의 핵심은 색상 분류예요. 색상만 제대로 써도 하루 일정을 한눈에 파악할 수 있어요.

💡
캘린더 자체를 색상별로 나누는 게 더 편해요

왼쪽 패널 → "다른 캘린더 추가" → 업무용, 개인용, 건강 등으로 캘린더 자체를 분리하면 색상이 자동으로 구분돼요.

🔄 반복 일정 & 알림 설정

매주 월요일 회의, 매일 운동 같은 루틴은 반복 일정으로 만들어두면 매번 입력할 필요가 없어요.

1
일정 생성 → "반복 안 함" 클릭

매일, 매주, 매월, 매년, 맞춤 설정까지 가능. "매주 월·수·금" 같은 복잡한 반복도 설정 가능해요.

2
알림 2개 설정이 효과적

30분 전 알림 + 5분 전 알림 두 개 설정. 처음 알림에 준비하고 두 번째 알림에 이동하면 절대 늦지 않아요.

3
기본 알림 시간 변경하기

설정 → 이벤트 설정 → 알림에서 기본값 변경 가능. 매번 설정 안 해도 돼요.

🎥 구글 Meet 링크 자동 생성

일정 만들 때 "Google Meet 화상 회의 추가" 버튼 하나만 누르면 링크가 자동 생성돼요. 초대받은 사람들에게 링크가 자동으로 공유됩니다.

✅
Zoom 링크도 연동 가능

구글 캘린더에 Zoom 앱을 연동하면 일정 생성 시 Zoom 링크도 자동 생성돼요. 구글 Workspace Marketplace에서 "Zoom" 검색 후 설치.

📡 다른 캘린더 구독하기

🇰🇷
한국 공휴일 자동 표시

왼쪽 패널 → "다른 캘린더" → + → "관심 캘린더 찾아보기" → 대한민국 선택. 공휴일이 자동으로 표시돼요.

👥
팀원 캘린더 보기

같은 구글 워크스페이스 사용 시 팀원 이메일로 캘린더 구독 가능. 회의 잡을 때 빈 시간 바로 확인.

🌐
URL로 외부 캘린더 구독

"다른 캘린더" → + → "URL로 추가". 스포츠 경기 일정, 드라마 방영일 등 외부 ics 파일도 구독 가능.

⚡ 숨겨진 꿀기능 5가지

1
드래그로 일정 이동·조정

일정을 드래그해서 다른 날로 이동하거나 하단을 드래그해서 시간을 늘릴 수 있어요. 마우스로 직관적으로 조정 가능.

2
단축키 활용 — C키로 빠른 일정 생성

캘린더에서 C 누르면 새 일정 창이 열려요. T는 오늘로 이동, 1·2·3·4·5는 보기 변경(일·주·월·연·일정목록).

3
이메일에서 자동 일정 추가

Gmail에서 항공권, 호텔 예약 이메일을 받으면 구글이 자동으로 캘린더에 일정을 추가해줘요. 설정 → 이벤트에서 켜기.

4
일정에 파일·링크 첨부

일정 생성 시 구글 드라이브 파일이나 링크를 첨부할 수 있어요. 회의 전 자료를 미리 첨부해두면 참석자들이 바로 확인 가능.

5
시간대 다르게 표시

설정 → 시간대에서 보조 시간대 추가 가능. 해외 팀이나 재택 원격 근무 시 두 나라 시간을 동시에 확인할 수 있어요.

🎯 핵심 요약
  • 색상 분류 — 업무·회의·개인 등 카테고리별로 색상 지정
  • 반복 일정 + 알림 2개(30분 전 + 5분 전) 설정
  • Google Meet 링크 자동 생성으로 회의 준비 시간 절약
  • 한국 공휴일 구독 + 단축키(C·T·1~5) 활용
  • 이메일 자동 일정 추가 켜두면 예약 놓칠 일 없음
