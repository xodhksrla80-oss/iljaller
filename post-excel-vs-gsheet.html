⚙️ 핵심 기능 비교

두 툴 모두 스프레드시트의 기본 기능은 동일해요. 차이는 고급 기능과 처리 속도에서 갈립니다.

기능📊 엑셀📗 구글 시트
기본 함수✓ 동일✓ 동일

📊
데이터 분석은 엑셀이 여전히 강해요

10만 행 이상 대용량 데이터, 복잡한 피벗, VBA 자동화가 필요하다면 엑셀이 압도적입니다. 재무·회계팀이 엑셀을 고집하는 이유예요.

🤝 협업·공유 — 구글 시트 압승

협업 측면에서는 구글 시트가 설계 철학 자체가 다릅니다. 엑셀은 개인 문서를 공유하는 개념이고, 구글 시트는 처음부터 공동 작업을 위해 만들어졌어요.

협업 기능📊 엑셀📗 구글 시트
실시간 공동 편집OneDrive 필요✓ 기본 제공
링크 공유설정 복잡✓ 1초
댓글·메모기본 수준✓ 강력
버전 기록수동 저장✓ 자동 무제한
오프라인 사용✓ 기본설정 필요
✅
구글 시트 최고의 장점 — 자동 저장 + 버전 기록

실수로 데이터를 지워도 걱정 없어요. 파일 → 버전 기록 → 버전 기록 보기에서 과거 어떤 시점으로도 복원 가능합니다. 저장 버튼이 없어도 돼요.

💰 가격·접근성

📊
엑셀 — Microsoft 365 구독 월 9,900원~

개인 플랜 기준. 회사에서 라이선스 제공하는 경우 무료. 일회성 구매(영구 라이선스)는 30만원대.

📗
구글 시트 — 완전 무료 (구글 계정만 있으면)

구글 드라이브 15GB 무료 포함. 추가 저장공간이 필요하면 Google One 구독 (월 2,900원~).

💸
개인·스타트업이라면 구글 시트가 답

비용 없이 협업까지 되는 구글 시트는 스타트업과 프리랜서에게 최적이에요. 직장에서 엑셀을 써야 한다면 집에서는 구글 시트로 연습해두는 것도 좋아요.
