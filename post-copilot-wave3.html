🚀 Wave 3란 무엇인가 — 'AI 에이전트 시대'의 시작

Microsoft가 2026년 3월 9일 Microsoft 365 Copilot Wave 3를 공식 발표했어요. Wave 1, Wave 2까지만 해도 'AI가 Office 앱에 녹아든다' 정도였는데, Wave 3는 이제 AI가 직접 실무를 '실행'하는, 말 그대로 에이전트 시대로 나아간다는 선언이나 다름없었습니다.

지금까지의 Copilot은 "이 이메일 요약해줘" 같이 사용자가 뭔가를 시키면, 단순하게 답만 내놓는 방식이었죠. 그런데 Wave 3의 진짜 포인트는 조금 달라졌습니다. 이제는 Copilot이 사용자의 목표만 전달받으면 알아서 단계별로 계획을 짜고 실행까지 스스로 해냅니다. 예를 들면, 이메일 분류나 문서 초안 작성, 회의록 정리처럼 반복되는 업무를 굳이 사람의 추가 지시 없이 자동으로 처리해준다고 하니 꽤 놀랍더라고요.

💡
Wave 3의 핵심 변화

이제 AI가 단순히 '답'을 던져주는 걸 넘어서서, 실제로 '업무'를 처리해주는 쪽으로 진화했어요. 사용자가 큰 방향(목표)만 알려주면, 디테일한 과정부터 결과까지 다 알아서 챙기는 식이죠.

특히 매일 반복되는 일에 치이는 직장인 분들께는 이 변화가 정말 실질적으로 다가올 것 같아요. 하루에 쌓이는 이메일 정리, 매주 보고서 뼈대 만들기, 회의 끝나고 액션 아이템 정리처럼 번거로운 루틴 업무들이 자연스럽게 자동화되는 셈이니까요.

📦 Agent 365 — 2026년 5월 출시되는 새 플랜

Wave 3에서 가장 주목받는 상품 중 하나가 바로 Agent 365예요. 2026년 5월 1일 정식 출시 예정이고, 기존 Microsoft 365 구독에 AI 에이전트 기능이 더해진 상위 플랜이라고 보시면 됩니다.

플랜주요 기능가격(예상)
M365 Business Basic기본 앱 + 이메일기존 동일
M365 CopilotAI 답변 보조기존 동일
Agent 365자율 에이전트 실행월 +$15
M365 E7엔터프라이즈 풀패키지별도 견적

Agent 365에서 정말 특이한 점은 자율 에이전트 실행이에요. 예를 들면 "매주 월요일 오전에 지난주 팀 이메일만 싹 정리해서 보고서로 만들어줘"라고 한 번만 설정하면, 그 뒤로는 자동으로 이 작업이 반복됩니다. 사람 손을 완전히 안 타는 거죠!

📌
Agent 365 출시 일정

공식 출시일은 2026년 5월 1일로 발표됐고요, 이미 일부 기업에서는 얼리 액세스 형태로 먼저 써보고 있다고 하네요.

🔗 Copilot Cowork — Claude 기반 장시간 자동화

이번 Wave 3에서 가장 혁신적으로 느껴지는 기능 중 하나가 바로 Copilot Cowork예요. Microsoft가 Anthropic과 손잡고 Claude라는 모델을 활용해서 만든 건데요, 일반 Copilot이 단일 작업(예를 들면 이메일 1통 요약)에 강했다면, Copilot Cowork는 진짜 장시간, 복잡한 연속 업무(경쟁사 10곳 조사 → 비교표 작성 → PPT 초안까지)를 한 번의 지시로 처리해줘요.

1
장시간 멀티스텝 작업

수십 분~몇 시간 걸리는 길고 복잡한 리서치나 보고서, 데이터 분석도 통째로 맡겨버릴 수 있습니다. 결과만 받아보면 되니 꽤 편리해요.

2
Anthropic Claude 기반

긴 문서 처리와 정확한 글쓰기 능력이 월등한 Claude 모델을 사용해서 문서 품질도 확실히 좋아졌어요.

3
현재 리서치 프리뷰 단계

2026년 3월 기준, 일부 기업 고객이 미리 써보고 있는 단계입니다. 곧 더 널리 만나볼 수 있지 않을까 기대됩니다!

✅
실제 활용 예시

"올해 1분기 영업 데이터를 분석하고 경쟁사 현황까지 비교한 뒤, 임원 보고용 슬라이드 5장 만들어줘" — 요즘 이런 요청 한 번이면 전체 작업이 뚝딱 해결됩니다.

📊 Word·Excel 인앱 Copilot — 실무에 이렇게 써보세요

Wave 3 버전부터 Word와 Excel에 바로 내장된 Copilot 기능이 훨씬 강력해졌어요. 예전엔 Copilot을 별도 창에서 실행해야 했는데, 이제는 문서 작업을 하다가 바로 원하는 자리에 AI 기능을 쓸 수 있습니다.

Word에서는 어떤 점이 달라졌을까요? 예를 들어, 긴 보고서를 쓰다가 중간에 막히는 파트가 생기면, 그 자리에서 Copilot에게 "이 단락 다음 내용, 논리 흐름에 맞게 이어서 써줘"라고 바로 요청할 수 있어요. 또, 각종 파일에 흩어진 정보를 한 번에 통합해서 보고서로 만드는 게 예전보다 훨씬 쉬워졌답니다.

Excel 쪽도 만만치 않아요. 복잡한 재무 모델도 이제는 자동으로 뚝딱 만들어집니다. "지난 3년 매출 데이터로 내년 예측 모델 만들어줘"라고만 치면, 필요한 수식이랑 차트까지 다 챙겨서 결과물을 내줍니다.

⚡
Excel Copilot 주요 기능 한눈에 보기

재무 예측 모델 만들기, 이상값 자동 탐지, 자연어로 피벗 테이블 구성 — 덕분에 엑셀 고수 아니어도 복잡한 데이터 분석이 부담 없어요.

Word Copilot 처리 가능 문서 길이
300페이지
Wave 3로 오면서 긴 문서 분석·요약 기능이 훨씬 강해졌다는 점, 확실히 체감됩니다

🛠 지금 바로 써볼 수 있는 3가지 기능

Wave 3에 추가된 모든 기능이 오늘 당장 쓸 수 있는 건 아니지만, Microsoft 365 정식 구독자는 지금도 바로 활용할 수 있는 Copilot 기능이 있거든요.

1
✉️ Outlook Copilot — 이메일 답장 초안 작성

받은 이메일 내용을 바탕으로 자동으로 답장 초안을 만들어줍니다. Outlook에서 Copilot 버튼만 누르고 '초안 작성'을 선택하면 바로 쓸 수 있어요.

2
🎤 Teams Copilot — 회의 요약 & 액션 아이템 정리

Teams에서 회의가 끝나면 Copilot이 자동으로 회의 요약과 할 일 목록까지 만들어줘요. 회의 중간에 Copilot 아이콘만 누르면 바로 활성화됩니다.

3
📝 Word Copilot — 새 문서도 초안부터 자동 생성

Word에서 새 문서를 열고 'Copilot으로 초안 작성'을 선택해 주제와 분량만 입력하면, 순식간에 초안을 뽑아줍니다. 초안 고민, 이제 끝이죠!

🔑
사용 조건

Microsoft 365에서 Business Standard 이상 플랜을 사용하고 있고, 관리자 권한으로 Copilot을 활성화해야 해요. 개인 플랜은 copilot.microsoft.com에서 무료 버전을 써볼 수도 있습니다.

  • Agent 365는 2026년 5월 출시 예정 — 자율 에이전트 실행 기능 추가 (월 +$15 예상)
  • Copilot Cowork는 Anthropic Claude 기술로 장시간 복잡한 작업도 처리 가능 (현재 프리뷰 중)

  • 오늘 당장 쓸 수 있는 기능: Outlook 이메일 초안, Teams 회의 요약, Word 빈 문서 초안 작성
